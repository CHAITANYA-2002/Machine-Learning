# Machine Learning Projects

A curated collection of machine learning, computer vision, NLP, forecasting, and data preprocessing experiments. The repository is organized as a project lab: each folder explores a specific modeling idea, dataset, or end-to-end workflow.

## Project Index

| Folder | Focus |
|---|---|
| `001_Cervical_Cancer_Predection_with_ML/` | Medical risk prediction with classical ML workflows. |
| `Bluetooth Sale Predictor/` | Sales prediction and forecasting experiments. |
| `CaliforniaHousePricePredictor/` | Regression workflow for California housing prices. |
| `Covid 19 predictor/` | Time-series and predictive analysis experiments. |
| `FakeNewsClassifier/` | NLP classification for misinformation detection. |
| `Gait trajectory/` | Gait trajectory modeling and analysis. |
| `ImageCRC/` | Image classification experiments. |
| `ManWomanClassifier/` | Image-based classification workflow. |
| `background_remover/` | Image segmentation and background removal exploration. |
| `image_enhancer/` | Image enhancement and super-resolution experiments. |
| `nifty/` | Stock-market prediction experiments. |
| `voiceshopping/` | Voice-assisted shopping prototype work. |

## Skills Demonstrated

- Data cleaning, feature engineering, and exploratory analysis.
- Regression, classification, clustering, and forecasting workflows.
- Notebook-based experimentation with reusable Python scripts.
- Computer vision tasks such as classification, enhancement, and background removal.
- NLP workflows such as fake-news classification.

## Getting Started

```bash
git clone https://github.com/CHAITANYA-2002/Machine-Learning.git
cd Machine-Learning
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

Then open the target project folder and follow its notebook or script instructions.

## Recommended Review Path

1. Start with `CaliforniaHousePricePredictor/` for a clean regression example.
2. Review `FakeNewsClassifier/` for NLP classification practice.
3. Explore `background_remover/` and `image_enhancer/` for computer vision work.
4. Check `Gait trajectory/` for research-style experimentation.
